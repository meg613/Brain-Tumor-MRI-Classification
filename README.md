# Brain Tumor MRI Classification using Deep Learning

This project uses deep learning and transfer learning techniques to classify brain MRI images into different tumor categories using PyTorch and ResNet18.

## Features
- MRI image classification
- Transfer learning using ResNet18
- Image preprocessing
- Model training and evaluation
- Prediction visualization

## Technologies Used
- Python
- PyTorch
- Torchvision
- Google Colab
- Matplotlib

## Dataset Classes
- Glioma
- Meningioma
- Pituitary Tumor
- No Tumor

## Model
A pretrained ResNet18 model was fine-tuned for multi-class brain tumor MRI classification.

## Results
The model achieved approximately 78.25% test accuracy.
